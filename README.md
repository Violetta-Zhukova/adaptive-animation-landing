# Адаптивный лендинг с анимациями

Проект по созданию современного адаптивного лендинга с продвинутыми анимациями и интерактивными элементами:

- Адаптивность (резиновый контент, плавное масштабирование элементов, медиа-запросы)
- Использование CSS-переменных
- Анимации и интерактив (SVG анимации, transition для всех интерактивных элементов, CSS фильтры, hover/focus/active состояния элементов)

https://violetta-zhukova.github.io/adaptive-animation-landing/
